Cuke4Nuke Intellisense for [Cuke4VS]

Mar 23rd, 2010

Needed to blog about this fantastic tool addin for VS2008 that gives intellisense for .feature files

http://github.com/henritersteeg/cuke4vs

This gives intellisense as follows:

 

It also gives us the option to add a .feature file when adding a new item to a solution, as follows:

 

 

One last beautiful feature of this tool is that when in the feature file and on a scenario step, then by pressing F12 you will be taking to the correct method in the class where it is matched.

The nice thing about this tool is that its free. Im not sure if there are any plans to incorporate this into VS2010 but if not id love to help out with this.

 

Anyone who uses cuke4nuke from visual studio should use this little adding
